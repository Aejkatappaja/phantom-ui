## Setup

<Tabs>
  <TabItem label="npm">
    ```sh
    npm install @aejkatappaja/phantom-ui
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```sh
    pnpm add @aejkatappaja/phantom-ui
    ```
  </TabItem>
  <TabItem label="yarn">
    ```sh
    yarn add @aejkatappaja/phantom-ui
    ```
  </TabItem>
  <TabItem label="bun">
    ```sh
    bun add @aejkatappaja/phantom-ui
    ```
  </TabItem>
</Tabs>

No extra TypeScript setup needed. Svelte picks up types from `HTMLElementTagNameMap` automatically.

## Usage

```svelte
<script lang="ts">
  import "@aejkatappaja/phantom-ui";
  import { onMount } from "svelte";

  let user = $state(null);
  let loading = $state(true);

  onMount(async () => {
    user = await fetchUser();
    loading = false;
  });
</script>

<phantom-ui {loading}>
  <div class="card">
    <img src={user?.avatar ?? "/placeholder.png"} alt="avatar" class="avatar" />
    <h3>{user?.name ?? "Placeholder Name"}</h3>
    <p>{user?.bio ?? "A short bio goes here."}</p>
  </div>
</phantom-ui>
```

Svelte handles boolean attributes natively. `{loading}` is shorthand for `loading={loading}`, and Svelte removes the attribute when the value is falsy.
